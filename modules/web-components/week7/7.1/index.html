# PWA Project

# Future Features

Here are some features that are not currently in the PWA, but that you might want to add in the future as you enhance this app for your own portfolio.

  1. Ability to import files.
  2. Add a server-side script which is hosted online that can accept copies of the cached files and save them so that they can be accessed from other copies of the app on different devices.
  3. Add user accounts so that the data will be tied to a single account. Users need to log in before they can access that data. Data stored on the device and can be seen, but data on the server needs to be connected to a user account.
